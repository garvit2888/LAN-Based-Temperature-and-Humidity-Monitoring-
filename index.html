Project Scope

Ensuring Infrastructure Stability

The server rooms at Manipal University Jaipur are critical to the institution's digital infrastructure. Without automated monitoring, they face risks of overheating and moisture damage.

Our LAN-Based IoT Monitoring System mitigates these risks by providing:

  • 24/7 Continuous Monitoring
  • Automated Multi-channel Alerts (SMS, Call, Email)
  • Historical Data Analysis & Export

System Features

Comprehensive monitoring solution designed for reliability.

Real-time Monitoring

SHT35 sensors provide high-accuracy temperature and humidity readings instantly, displayed on both the web dashboard and local 16x2 I2C panels.

Automated Alerts

Immediate notifications via Twilio (SMS & Call) and Email when environmental thresholds are breached, ensuring rapid response.

Reliable Connectivity

Utilizes W5500 Ethernet Modules for stable LAN connectivity and Power over Ethernet (PoE) for streamlined deployment.

Data Analytics

MySQL-backed storage allows for detailed historical trend analysis via Grafana and CSV/Excel exports for reporting.

Technical Architecture

Built on robust hardware and modern software stacks.

Hardware

  • Microcontroller: ESP32
  • Ethernet: W5500 module
  • Sensor: SHT35
  • Display: I2C LCD display

Software

  • Frontend: Flutter
  • Backend: Python3, FastAPI
  • Database: SQLite, Influx
